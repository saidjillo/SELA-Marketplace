🔒 Maintenance Operations

Sensitive operations like database resets, data exports, and server restarts are intentionally not available from this UI for security reasons. These must be performed directly on the server terminal by an authorized system administrator.

📋 Available terminal commands:
🗑️
Reset database (wipe all data)
node reset-db.js
🔄
Restart the server
Ctrl+C  →  npm start
🌱
Seed demo vendor data
node seed-vendor-products.js
🔗
Backfill product slugs
node backfill-slugs.js
💡 Keep your server terminal accessible only to authorized personnel. Never share your .env file or admin credentials.
